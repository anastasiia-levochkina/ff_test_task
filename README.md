# Minesweeper - FriendlyFox Test Task

Classic Minesweeper implemented in Lua for the FriendlyFox engine.

## How to run
1. Place `TestScreen.lua` in the same folder as `FF_Sandbox.exe`
2. Launch `FF_Sandbox.exe`

## Gameplay
- Left click — open a tile
- Right click — place/remove a flag
- Click Restart to start a new game

## Field
15×8 grid, 20 mines
